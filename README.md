# 咖啡伴侣 Coffee Companion

> 用自然语言记录每一杯咖啡，让 AI 帮你读懂自己的口味。

**[立即体验 →](https://frestal.github.io/coffee-companion/)**

---

## 它是什么

大多数咖啡 App 要求你像专业人士一样填表：产区、处理法、风味轮……而大多数人喝咖啡时只是有一个感受。

咖啡伴侣反过来：**你说你的，AI 来翻译。**

"今天这杯有点像梅子汁，酸酸的但喝完嘴里还有点甜" —— 这就够了。伴侣会认真对待你说的每一个字，帮你把口语化的感受映射回专业风味体系，同时悄悄积累你独有的咖啡画像。

---

## 核心功能

### 自然语言记录
不用懂术语。拍一张豆袋照片，或者直接描述你喝到的感受，AI 帮你识别和理解。Claude Vision 会从豆袋图片中提取产区、处理法、品种等信息，自动填入记录。

### 每日晨间洞察
每天早上，根据你过去所有的咖啡记录，AI 生成一段个性化洞察——你最近偏向什么风味，有什么值得关注的变化。不是通用推荐，是真正属于你的观察。

### 调配处方（≥5条记录解锁）
积累了足够的记录后，AI 会根据你的口味偏好生成一张专属处方卡：适合你的产区、处理法、烘焙度、冲煮参数、风味关键词。一键复制，出示给咖啡馆，让店员帮你选豆。

### 惊艳时刻
喝到一杯让你眼睛一亮的咖啡？标记它。这些时刻会被单独收藏，成为你咖啡轨迹里最值得回味的切片。

### 风味图谱
所有记录中出现的风味词汇，自动聚合为可视化气泡图。哪些词频繁出现，就是你真正偏好的风味核心。

### 咖啡特征码
根据你的记录和偏好，生成一个专属特征码，格式如 `CF-FE-梅-007`。

- **风格**：产区猎人 / 风味探险家 / 惊喜收集者 / 深耕者
- **感受标签**：从你最常用的描述词提取
- **记录数**：见证你的成长轨迹

特征码可以出示给咖啡馆，让店员快速了解你的口味。

### 咖啡图鉴
内置精品咖啡知识库，覆盖全球主要产区、豆种、处理法、冲煮器具。不是枯燥的词典，而是帮你在喝的时候能看懂背景的参考。

## 产品理念

**记录应该是低摩擦的。** 打开 App，说出感受，关掉。不应该比点一杯咖啡更复杂。

**数据应该是你的。** 所有记录保存在本地 localStorage，不上传服务器，不注册账号，不绑定任何平台。API Key 也存在本地，由你掌控。

**AI 是伴侣，不是裁判。** 它不会告诉你喜欢"错误"的咖啡。它的角色是帮你更清楚地认识自己的口味，而不是引导你走向所谓标准答案。

## 技术架构

| 项目 | 说明 |
|------|------|
| 架构 | 单文件 PWA，无构建工具，无框架 |
| 数据 | 全量存储在 localStorage，无后端数据库 |
| AI | 直接调用 Anthropic API（claude-sonnet），需用户自备 API Key |
| 部署 | GitHub Pages，`git push` 即生效 |
| 离线 | Service Worker，主文件 Network First，静态资源 Cache First |
| 更新 | 新版本上线后用户自动无感刷新 |

## 快速开始

1. 打开 [frestal.github.io/coffee-companion](https://frestal.github.io/coffee-companion/)
2. 进入「我的」页面，填入你的 [Anthropic API Key](https://console.anthropic.com/)
3. 记录第一杯咖啡

无需注册，无需安装，支持"添加到主屏幕"作为 PWA 使用。

---

## 本地开发

```bash
git clone https://github.com/frestal/coffee-companion.git
cd coffee-companion
# 用任意 HTTP 服务器打开，例如：
npx serve .
```

所有逻辑在 `coffee_v6.html` 中，单文件可直接编辑。
